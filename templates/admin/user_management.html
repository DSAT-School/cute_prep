User Management Tips

  • Edit: Click the edit icon to modify user details inline
  • Change Role: Assign or change a user's role with one click
  • Impersonate: Login as any user for 10 minutes to troubleshoot issues
  • Deactivate: Temporarily disable user access without deleting their data

  • Onboarding Toggle: Control whether new users can register on the platform
